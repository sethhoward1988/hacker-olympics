  • String Length: 72 characters
  • Repeating Pattern: aueeuaeauuae
  • Times Repeated: 6

This problem faced a few challenges, but depending on what language you used, some may or may not have needed to be overcome.

When the Fibonacci sequence got to a higher number, it exceeded the capacity of many languages to execute simple math with precision. For example, in JavaScript, the max value with precision is just over 9000000000000000. The 78th Fibonacci term is just under it at 8944394323791463, while the 79th exceeds it at 14472334024676218. Essentially, when trying to modulo numbers larger then JavaScripts max, there were errors which threw off the entire code. This however can be resolved by adding a BigInt library, of which many languages already come with. If you chose Python, you didn't even have to worry because the language is precise enough as it is.

For everyone who is curious, below is a well documented and easy to follow script that explains sequentially how each process was executed. Many, if not all, of the entrants combined these steps together or forwent steps that they deemed unneccessary. As long as the output was correct, and computed via a computer, then their answers were also correct. 

var a = BigInteger(0);
var b = BigInteger(1);
var fib = [a,b];
var stripped = [];
var alphabetized = [];
var answer = [];
var alphabet = ['a','b','c','d','e','f','g','h','i','j','k','l','m','n','o','p','q','r','s','t','u','v','w','x','y','z'];
var vowels = ['a','e','i','o','u'];

//Continue the fibonacci sequence out to the 500th term
for(var i = 1;i < 499; i++){
    var newFib = fib[i].add(fib[i-1]);
    fib.push(newFib);
};

//Strip out all of the odd indexes and place into a new array
for(var i = 0; i < fib.length; i++){
    if(i % 2 == 0){
        stripped.push(fib[i]);
    }
}

//Find the modulo of each number in the array, and match it with the corresponding letter in the alphabet array
for(var i = 0; i < stripped.length; i++){
    var modulo = stripped[i].remainder(26);
    if(modulo == 0){
        modulo = 26;
    }
    alphabetized[i] = alphabet[modulo - 1];
}

//Compare each letter in the array to see if it is in the vowel array, pull out anything that doesn't match
for(var i = 0; i < alphabetized.length; i++){
    if(vowels.indexOf(alphabetized[i]) != -1){
        answer.push(alphabetized[i]);
    }
}

//Concatenate the answer array with an empty string separator and log to the console
console.log(answer.join(''));
